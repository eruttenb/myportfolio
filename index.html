Project - Q&A Forum

This is the final project for my Software Engineering class, for which we made a web-based application for asking questions and posting answers.

What I Learned

Almost everything used for this project was something new I learned in that class. The Agile-Scrum methodology, Flask, CSS, and Python were all new concepts/languages. I benefited in this learning process by collaborating with more experienced team members, who gave me a push in the right direction. We were all lucky enough to supplement each others shortcomings and work efficiently on the application.

How We Made It

We collaborated in an Agile-Scrum environment and utilized GitHub for group work. The application was developed using Flask as a framework, CSS for front-end, and Python for back-end.

Challenges

Later on in the project we realized that our additional features were too complicated, and we had too many of them. For example, a search bar was difficult to implement on our home page, so we ended up getting rid of it. Working on a major coding project in a team environment also lead to some technical difficulties, especially with everyone having different schedules and availabilities; sometimes changing little things would mess up a larger section of code since you wouldn't always understand someone's thought process from what they added.
